# 🌍 Lugares que Queremos Conhecer

Este projeto foi desenvolvido como parte de uma atividade prática na aula de **DevOps**. O objetivo principal foi exercitar o fluxo de trabalho colaborativo utilizando o Git e o GitHub, com foco especial na ferramenta de **Pull Request (PR)**.

## Sobre o Projeto

O repositório funciona como um mural coletivo onde cada integrante do grupo compartilhou seus sonhos de viagem e pontos turísticos de interesse. Através desta dinâmica, pudemos simular um ambiente real de desenvolvimento onde múltiplos contribuidores enviam alterações para um mesmo projeto.

### Objetivos Principais:
* **Prática de DevOps:** Entender o ciclo de vida de uma alteração de código, desde a criação de branches até o merge final.
* **Gestão de Pull Requests:** Aprender a abrir, revisar e aceitar contribuições de outros desenvolvedores.
* **Colaboração:** Compartilhar informações pessoais sobre sonhos e destinos turísticos através de HTML e CSS.

## Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **Git & GitHub** (Foco em colaboração e controle de versão)
